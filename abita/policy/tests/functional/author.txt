Log in as the portal owner::

    >>> from plone.app.testing import TEST_USER_NAME, TEST_USER_PASSWORD
    >>> browser.login(TEST_USER_NAME, TEST_USER_PASSWORD)

Go to portal and check that there are no link to dashboard::

    >>> browser.open('/author')
    >>> 'id="content-core"' in browser.contents
    False

    >>> author_url = '/author/{0}'.format(TEST_USER_NAME)

Go to author_url::

    >>> browser.open(author_url)
    >>> 'id="edit-bar"' in browser.contents
    False
    >>> 'id="content-core"' in browser.contents
    False

Log out and do the same thing::

    >>> browser.getLink('Log out').click()

    >>> browser.open('/author')
    >>> 'id="content-core"' in browser.contents
    False

    >>> browser.open(author_url)
    >>> 'id="edit-bar"' in browser.contents
    False
    >>> 'id="content-core"' in browser.contents
    False
